# License inventory

- `Apache-2.0.txt` covers the Rust wrapper and the ManT-authored C shim.
- `mandoc-1.14.6.txt` is the upstream license inventory for the complete
  vendored libmandoc 1.14.6 source tree.

The upstream inventory identifies the predominantly ISC-licensed mandoc
sources and the selected compatibility files distributed under BSD-2-Clause
and BSD-3-Clause licenses.  Individual vendored source files retain their
copyright notices.
